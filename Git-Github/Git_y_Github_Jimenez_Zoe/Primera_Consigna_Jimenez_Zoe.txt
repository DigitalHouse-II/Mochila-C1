a. ¿Quién inventó el sistema de control de versión Git y por qué?
Git fue creado por Linus Torvalds en 2005 con el propósito de ser
utilizado en el desarrollo de Linux.
b. ¿A quién pertenece actualmente Github y por qué?
Actualmente Github pertenece a Microsoft, que lo compró en el 2018.
Desde que Satya Nadella fue nombrado CEO de Microsoft se adoptó
en esta empresa una estrategia más abierta al mundo del software
libre y a la integración con otras plataformas y servicios, y esta compra
fue un paso hacia esa dirección.
c. ¿Hay otra forma que no sea la terminal para trabajar con
Github?
No hay otra manera de trabajar con Github si no es con git, o bien
integrando git a otro programa como por ejemplo Visual Studio Code.
