git init: Crea repositorio local
git config (user.name)(user.email): Provee usuario y email de GitHub para linkear el repositorio
git add: Agregar archivos al tracking de github. Con "." agregamos todo el repositorio. "nombrearchivo" agrega archivo
git commit -m "": Confirmamos cambios realizados para luego subir a la nube. Entre "" ponemos descripcion del mismo

git remote add origin "linkrepositorioengithub": Establece conexion con repositorio en la nube
git push origin main: Subimos archivos a repositorio remoto
git push origin "nombrerama": Subimos archivos a rama especifica de dicho repositorio
git pull origin main: Descarga archivos alojados en la nube
git pull origin "nombrerama": Descarga archivos alojados en rama especifica
git clone "linkrepositorio": Clona repositorio remoto a carpeta especifica
